
객실 예약관리 메뉴

1. 객실 예약 / 2. 예약 수정 / 3. 객실 등급별 내용 / 4. 연간회원 등록

1. 객실 예약
고객명 입력(고객번호 매칭 후 연간회원등급 확인 후 등급란에 자동입력, 해당없으면 일반등급) , 객실 등급 선택, 예약 시작날짜, 예약 마지막날짜, 숙박인원 입력

2. 예약 수정
고객명 입력
(1) 예약 시작날짜 변경 / (2) 예약 마지막날짜 변경 / (3) 객실등급 변경 / (3) 인원 변경 / (4) 예약 취소

3. 객실 등급별 내용
(1)스탠다드 룸 / (2)디럭스 룸 / (3)패밀리 룸 / (4)스위트 룸
스탠다드 룸 - 방개수, 침대사이즈, 침대갯수, 가격
디럭스 룸 -
패밀리 룸 - 
스위트 룸 - 

4. 연간회원 등록

일반 - 
골드 - 예약 할인, 동반객 할인, 무료객실이용권 갯수  
플래티늄 -

      예약 테이블
칼럼 고객명(기본키)    예약시작날짜    예약마지막날짜    객실등급명(외래키)    숙박인원    예약접수일    고객연간등급(외래키)
      
      객실 등급 테이블
칼럼 객실등급명(기본키)    방개수    침대사이즈    침대갯수    최소예약인원    가격
